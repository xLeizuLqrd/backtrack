# backtrack

**Приложение для замещения исполняемых файлов**

---

## Описание

backtrack — это приложение на python с графическим интерфейсом, предназначенное для замены EXE-файлов с последующим необратимым удалением оригиналов. Программа сохраняет метаданные файла (дату создания и изменения), что делает замену незаметной для операционной системы.

---

## Функциональные возможности

- **Замещение** выбранного EXE-файла с сохранением оригинального размера
- **Сохранение временных меток** файла (дата создания, изменения)
- **Автоматическое переименование** целевого файла
- **Переключение между светлой и тёмной темой** интерфейса
- **Поддержка русского и английского языков**

---

## Установка и использование

### Сборка и первый запуск

1. **Скачайте** последний релиз из раздела [Releases](https://github.com/xLeizuLqrd/backtrack/releases)
2. **Распакуйте** архив в удобную директорию
   > ⚠️ **Важно**: Файл `70b7v6lg.exe` **не входит** в дистрибутив. Вы должны добавить его самостоятельно. Это может быть любой исполняемый файл, переименованный в `70b7v6lg.exe`. Именно этот файл будет переименован в имя оригинального файла после завершения процесса замещения.
3. **Установка зависимостей**
```
pip install -r requirements.txt
```
4. Запустите сборку приложения build.bat
5. **Поместите** любой EXE-файл с именем `70b7v6lg.exe` в папку с backtrack.exe

![Расположение](/pictures/screen.png)

7. **Запустите** `backtrack.exe` (он будет находиться в папке ./dist/ после окончания сборки)
8. **Выберите** файл, который требуется заместить
9. **Готово!** выбранный файл исполняет уже не тот код, что раньше
10. **Для запуска** замененного `.exe` файла требуются права администратора (без них не запустится)
### Сборка выполняется один раз и в следствии Вам нужно будет запускать только backtrack.exe и заменять любые `.exe` файлы 
Изначально приложение создавалось для скрытия читов (.exe) от ручной проверки администраторами серверов
